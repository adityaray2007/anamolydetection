### Recommended Specs (For smooth, real-time performance)
To get the best experience where the VLM responds quickly and face recognition is instant, a laptop with a dedicated NVIDIA GPU is ideal.

- CPU: Intel Core i7 (10th Gen+) / AMD Ryzen 7
- RAM: 16 GB System RAM
- GPU: Dedicated NVIDIA entry-level gaming GPU (like an RTX 3050, 4050, or even an older GTX 1650) with at least 4GB of VRAM.
- Storage: NVMe SSD with 10 GB free space.


### Moving it to Windows
Since the project is written in Python, moving it is straightforward:

- Install Python 3.12 on Windows.
- Install Ollama for Windows from their website and run `ollama pull moondream`.
- Copy this folder over, create a virtual environment, and run `pip install -r requirements.txt`.
